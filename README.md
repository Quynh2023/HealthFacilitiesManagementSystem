## How to run
To run the project:

1. Turn on XAMPP to connect MySQL.
2. In Visual Studio Code: open the project 
3. In a terminal, to run APIs: npm i -> cd [api file path] -> nodemon api.js
4. Open new terminal, to run the website: npm start
